⚙️ Configuration Firebase
Suivez ces étapes une seule fois pour activer la synchronisation en temps réel pour toute la tribu.
1 Allez sur console.firebase.google.com → Créez un projet (ex: laww-tracker)
2 Dans le projet → Firestore Database → Créer une base de données → Mode production → Choisissez une région
3 Onglet Règles → Remplacez le contenu par :
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if true; } } }
→ Publiez
4 ⚙️ Paramètres du projet → Ajouter une application Web (</>) → Copiez le bloc firebaseConfig
5 Ouvrez ce fichier HTML avec un éditeur texte → Cherchez FIREBASE_CONFIG en haut → Remplacez les valeurs → Sauvegardez et re-uploadez sur GitHub
